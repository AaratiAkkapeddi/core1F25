Project 8: Web to Print

Due 11/18 9am

For this assignment, you'll work off of the class generative poster demo to create a printable webpage that gives users control over what is printed. Using what we learned in class, design a feature where users can customize some aspect of the page's appearance or content by using HTML inputs and javascript. This customization should align with your harmonic collection theme. For example, if your theme is furniture design, users might design their own chair by selecting options; if it's dancing with the stars, they might design their own 'paper doll'-style dance costumes by selecting from a dropdown of images; if it's nightlife, they might design their own cocktail recipe to be printed by selecting ingredients. The key is that inputs should give the user a meaningful choice that enhances their interaction with your themed content. Your page should include at least three different types of inputs (i.e. text, color, select dropdown, or more ) and at least three aspects that visually change when the inputs are changed.Please make sure to include a css class with the name 'colorbean' There should also be a 'print' button (as is in the demo) and a meaningful print media query in your css that helps style your page for printing and hides what you don't want to print.

Technical Requirements:

  • Figma design (submit in canvas)
  • Use normal @media queries to make sure your website works on both desktop and mobile
  • Use print @media queries to make sure your website prints how you want it to
  • At least three different types of form inputs
  • At least three different aspects that change when the user changes the inputs

Grading Criteria:

  • Conceptual depth (30%): Clear, intentional perspective shift that engages with device context
  • Design execution (30%): Both versions are polished, cohesive, and well-designed
  • Technical implementation (30%): Clean code, effective use of media queries, JS, semantic HTML
  • Integration with portfolio theme (10%): Successfully connects to your larger body of work
