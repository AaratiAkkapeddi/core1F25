HTML starter code cheatsheet

Every html file needs certain things like DOCTYPE, <head> and <body> tags. Below is starter code that you can use for when you begin a new HTML page. You will need to modify this code to your needs.

<!DOCTYPE html>
<html lang="en">
    <head>

        <!-- we need these two meta tags for non-latin characters and mobile compatibility -->
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0"> 

        <!-- title of your page (shows up in tab of browser) -->
        <title>Title of your page/project</title>

        <!-- Connecting a CSS file -->
        <link rel="stylesheet" href="nameofyourcssfile.css">

    </head>
    <body>

        <!-- content goes here -->
        
    </body>
</html>
