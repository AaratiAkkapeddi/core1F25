Holding Position

Create a multipage website (at least 2 pages) that connects to your Harmonic Collection theme. Each page should have enough content so that it scrolls. Include a navigation that stays visible as the user scrolls (use fixed or sticky positioning for this). You can choose the style and content, as long as it reflects your theme.

Due 9/30

  1. In figma: Submit a link in canvas to a figma design for your Holding Position project. The links should be interactive following the instructions we went over last week. Submitting a figma design is mandatory.
  2. In HTML & CSS code: Using HTML & CSS code your design into a functional website. Submit all of your project files (image assets, html, and CSS) here
